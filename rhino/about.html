About the game

A rhino is lost in New York. Can you find it?

In this program New York is a 8 x 8 grid and the user enters co-ordinates to guess the position of the rhino. The computer responds by saying how far away the user has to walk (along the grid lines) to find the rhino. This distance is recorded on the grid.

The aim is to minimise the number of guesses and this involves some strategic thinking, which will often be enchanced when the children are working in small groups.
