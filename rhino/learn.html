Controls

The aim of this game is to correctly guess the location of a rhino that is missing in New York city. In order to do that, the user has to select the co-ordinates of where they think the rhino is and click the "Guess" button.

To select the coordinates use the two sliders provided. The upper slider selects the X co-ordinate, similarly, the lower slider selects the Y co-ordinate. Each time the user moves the sliders the selected value will appear above the upper slider.

Once the user has clicked the "Guess" button, if the entered co-ordinates are corect the user will be notified, otherwise, the distance from the users guess to the rhino will be places at the location of the guess.

Score

The aim of this game is to try and find the rhino in as few guesses as possible. The player is encouraged to use strategic guessing. Additionally, younger players are encouraged to play in groups.
